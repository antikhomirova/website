# Anna Tikhomirova – Bioengineering Portfolio

[![Made with HTML/CSS/JS](https://img.shields.io/badge/made%20with-HTML%2FCSS%2FJS-ff69b4?style=flat-square)](https://developer.mozilla.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=flat-square)](https://opensource.org/licenses/MIT)

A personal portfolio website showcasing bioengineering research, medical device projects, and professional experience.  
Designed with a **Y2K aesthetic** – blending retro web design with modern functionality.

---

## ✨ Features

- **Y2K Visual Style** – Neon gradients, pixelated borders, glass‑morphic panels, custom cursor, animated ECG line.
- **Interactive Project Cards** – Click any project to open a modal with full description and links (YouTube, slides, etc.).
- **Smooth Scrolling & Scroll Animations** – Sections fade in as you navigate.
- **Fully Responsive** – Works on desktop and mobile.
- **Single File** – Everything is contained in `index.html`; no build steps required.
